
# Kyrgyzstan Migration Atlas

> **⚠️ Work in Progress**
>
> This repository is an evolving GIS and development portfolio project focused on migration, remittances, poverty, and rural development in Kyrgyzstan. New datasets, analyses, and visualizations will be added as the project develops. Current materials should be considered preliminary and subject to revision.

---

## Project Overview

Kyrgyzstan is one of the most remittance-dependent countries in the world. Labor migration has shaped household economies, demographic patterns, and development outcomes across the country for decades.

This project explores the relationships among migration, remittances, poverty, and infrastructure development in Kyrgyzstan through the use of GIS, publicly available datasets, and data visualization techniques.

The goal is not simply to create maps, but to better understand how population movements influence rural livelihoods and regional development.

---

## Research Questions

* How have migration patterns changed across Kyrgyzstan since 2013?
* Which oblasts experience the highest levels of outmigration and return migration?
* How are remittances associated with poverty reduction?
* Have remittances contributed to sustainable local development, or primarily increased household consumption?
* How have infrastructure investments influenced migration patterns?
* What do these trends imply for the future of rural communities in Kyrgyzstan?

---

## Geographic Focus

The project focuses primarily on:

* Kyrgyzstan (national level)
* Oblast-level comparisons
* Selected rayon-level case studies (future work)
* Southern Kyrgyzstan, including Osh Oblast and Nookat Rayon
