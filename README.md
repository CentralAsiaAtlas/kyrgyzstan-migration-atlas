# Kyrgyzstan Migration Atlas

This repository contains GIS data, maps, and analysis related to migration, agriculture, rural development, and economic change in Kyrgyzstan.

## Objectives

- Visualize migration patterns across Kyrgyzstan
- Develop reusable GIS datasets for researchers
- Integrate demographic, agricultural, and economic indicators
- Support research on rural development and post-Soviet transformation

## Current Contents

- Administrative boundaries (oblasts)
- QGIS project files
- Map outputs

## Planned Additions

- Rayon-level boundaries
- Migration statistics
- Remittance data
- Population data
- Agricultural indicators

## Data Sources

- National Statistical Committee of the Kyrgyz Republic
- IOM Kyrgyzstan
- Humanitarian Data Exchange (HDX)
- World Bank
